# Diagram Generation Prompts

## Diagram 1 — AWS FinOps Architecture

Create a polished enterprise AWS architecture diagram showing a DevOps FinOps workflow. Include Developer, Terraform Change, GitHub Pull Request, Atlantis / Terraform Plan, Infracost Estimate, Cost Delta Comment, Engineering Review, AWS Deployment, AWS Cost Explorer, Kubecost, Monthly FinOps Review and Optimization Backlog. Use AWS-inspired orange, dark navy and green accents, clean white background, modern executive presentation style, no clutter.

## Diagram 2 — Terraform PR Cost Visibility Flow

Create a clean process diagram showing how cost visibility is added to every Terraform pull request. Flow: Developer creates Terraform change, Pull Request, Terraform plan, Infracost estimate, PR cost delta comment, reviewer decision, approve or redesign, merge and deploy. Use professional DevOps style, GitHub and Terraform inspired colors, clear arrows, executive-ready formatting.
